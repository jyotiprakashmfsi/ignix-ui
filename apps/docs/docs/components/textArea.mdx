# üìÑ `AnimatedTextarea` Component Documentation

## Overview

`AnimatedTextarea` is a highly customizable, animation-driven textarea component built with **React** and **Framer Motion**. It supports various animation variants that affect both label and textarea appearance, enhancing UX through visual cues like glowing borders, character counts, line highlights, and more.

## üí° Features by Variant

Some variants include extra features:

| Variant            | Description                                                                 |
|--------------------|-----------------------------------------------------------------------------|
| `expandable`       | Auto-resizes based on content height.                                       |
| `smoothExpand`     | Same as `expandable` but with animation smoothing.                          |
| `characterCount`   | Displays live character count (`value.length / 500`).                       |
| `lineHighlight`    | Highlights a line based on character count.                                 |
| `particleField`    | Shows a looping animation of floating particles around the textarea.        |
| `cosmic`, `wave`, `hologram`, etc. | Continuous animated effects for futuristic feel.            |

## üß† Internal Behavior

- Uses `Framer Motion`'s `motion.label` and `motion.textarea` for animated label and field transitions.
- Tracks `focus`, `blur`, and `value` changes for triggering animation states.
- `textareaRef` is used for calculating dynamic height (`scrollHeight`) for expanding variants.
- Extra UI elements like `div`s or labels are conditionally rendered based on the active variant.
