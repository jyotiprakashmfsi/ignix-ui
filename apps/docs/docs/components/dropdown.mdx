# Dropdown Component

A customizable and animated dropdown component built with Radix UI, Framer Motion, and `class-variance-authority` (CVA).

- Smooth animations (fade, scale, slide, flip, etc.)

## ðŸŽ¬ Animations

Available animation variants:

- `default`: Slight scale + vertical slide + fade
- `fade`: Simple fade in/out
- `scale`: Scale with opacity
- `slide`: Slide down with opacity
- `flip`: Flip animation

## ðŸ”§ Usage

```tsx
import { Dropdown, DropdownItem } from "@/components/ui/dropdown";
import { Button } from "@/components/ui/button";

<Dropdown
  trigger={<Button>Open Menu</Button>}
  animation="slide"
  bg="glass"
  rounded="md"
>
  <DropdownItem onSelect={() => alert("Profile")}>Profile</DropdownItem>
  <DropdownItem onSelect={() => alert("Settings")}>Settings</DropdownItem>
  <DropdownItem onSelect={() => alert("Logout")}>Logout</DropdownItem>
</Dropdown>
```
