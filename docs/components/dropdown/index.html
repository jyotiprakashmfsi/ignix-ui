Dropdown

Overview​

The Dropdown component provides a flexible and customizable way to create dropdown menus with various styling options, animations, and sizing variants. Built with Radix UI and Framer Motion, it offers smooth animations and accessible functionality out of the box.

Preview​

  • Preview
  • Code
import { Dropdown, DropdownItem } from '@mindfiredigital/ignix-ui';

function DropdownDemo() {
  return (
    <Dropdown
      trigger={<Button>Open Menu</Button>}
      animation="default"
    >
      <DropdownItem>Profile</DropdownItem>
      <DropdownItem>Settings</DropdownItem>
      <DropdownItem>Logout</DropdownItem>
    </Dropdown>
  );
}

Installation​

  • npm
  • Yarn
  • pnpm
npx @mindfiredigital/ignix-ui add dropdown
yarn @mindfiredigital/ignix-ui add dropdown
pnpm @mindfiredigital/ignix-ui add dropdown

Usage​

Import the components:

import { Dropdown, DropdownItem } from '@mindfiredigital/ignix-ui';

Basic Usage​

function BasicDropdown() {
  return (
    <Dropdown trigger={<Button>Click me</Button>}>
      <DropdownItem>Option 1</DropdownItem>
      <DropdownItem>Option 2</DropdownItem>
      <DropdownItem>Option 3</DropdownItem>
    </Dropdown>
  );
}

Props​

Dropdown Props​

PropTypeDefaultDescription
childrenReactNode-The content of the dropdown menu
triggerReactNode-The element that triggers the dropdown
animation'default' | 'fade' | 'scale' | 'slide' | 'flip''default'Animation style for the dropdown
size'sm' | 'md' | 'lg''md'Size of the dropdown menu
rounded'none' | 'sm' | 'md' | 'full''md'Border radius of the dropdown menu
bg'default' | 'dark' | 'transparent' | 'glass' | 'gradient' | 'primary''default'Background style of the dropdown
classNamestring-Additional CSS classes

DropdownItem Props​

PropTypeDefaultDescription
childrenReactNode-The content of the dropdown item
classNamestring-Additional CSS classes
...propsReact.ComponentProps<typeof DropdownMenu.Item>-Any additional props from Radix UI's DropdownMenu.Item

Customization​

Custom Styling​

You can customize the appearance using the provided props or by passing custom class names:

<Dropdown
  className="my-custom-dropdown"
  trigger={<Button>Custom Styled</Button>}
  size="md"
  rounded="sm"
  bg="glass"
>
  <DropdownItem className="hover:bg-blue-100">
    Custom Item
  </DropdownItem>
</Dropdown>

Custom Animation​

While the component comes with predefined animations, you can extend it with custom animations using the className prop and Tailwind CSS classes:

<Dropdown
  className="transition-all duration-300"
  trigger={<Button>Custom Animation</Button>}
>
  <DropdownItem>Animated Item</DropdownItem>
</Dropdown>
